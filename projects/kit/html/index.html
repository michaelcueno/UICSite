User Interface Project Two

A Family (HouseHold) Organization Application developed in QML @ UIC

DESCRIPTION

KIT (Kitchen Interface Technology) is a project desinged for a typical family. It is meant to be installed in the kitchen and seemlessly augment the user experience in the kitchen. The interface attempts to dissapear and allow the user to get what they need fast. To implement this, we started with the idea of modeling the kitchen from within the app. Thus we have our main screen featuring a 'window' where the outside represents the current weather, the blinds each correspond to a specific user and contains information about that users' accounts, and the windowsill is full of icons that go to our main applications. These window-sill icons each launch an app that anyone in the family can use and has no user specific information. These apps include the Music player and the weather. Certainly if there is some music playing, anyone in the household should be able to access the music app quickly without needing to sign in first.

General Interactions

The app is centered around the touch screen interface. Thus swiping and tapping are common themes in the app. Going along with our kitchen interface, we get to our user specifc applications. These are the main window after tapping on a user blind, and the three other views that become accessable upon 'logging-in'. These views are accessable via a two finger swipe to the left, right and down which goto the cooking, fridge and cabinet views respectively.
