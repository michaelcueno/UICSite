Download

Compiled App

The app was compiled for linux (Ubuntu 12.04) and should work if you have the correct qt libraries installed on your system.
You can download the tar ball here
To install, run
tar -xvf kit.tar.gz
cd kit
./src
(Note: the app runs in 1920 x 1080 fullscreen) There is no quit since the app was meant to run on dedicated hardware, so you will need to kill via the terminal:
ps aux | grep src
kill -9

IMPORTANT!!

Make sure to change the global root directory to where you downloaded the application. Change the variable "root_dir" found here. This is needed so that the application can access the images properly.

Compile from source

To compile from source, download the source from the githup repo
git clone https://github.com/mcueno2/kit.git
and open src.pro in qtcreator
