User Interface Project One

A Weather Station application developed in QML @ UIC

DESCRIPTION

The goal for this project was to develop a 1200X800 pixel, landscape application for viewing the local weather. The app also called for functionalities such as adding new cities, multiple language support, and graphical representation of the weather.

General Interactions

The main way the user interacts with the application is by swiping the hours bar, as the user swipes towards later hours, the sun/moon will fall and the main stats (temp, condition..) will update to the currently highlighted hour. The user can also tap on a day of the week to go to that days forecast immediately.
